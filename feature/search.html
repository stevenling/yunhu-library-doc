# 搜索书籍

# 功能

可以通过书名或者作者名来进行搜索，支持模糊搜索。

# 搜索书籍模块设计

一个输入框，一个搜索按钮，搜索结果在底下，也提供分页，因为一个作者可能写了很多书籍。

不同标签可能有同一本书，比如《三体》属于科幻标签，也属于文学标签，因此搜索请求后，前端会进行去重。

搜索结果也提供下载接口。
