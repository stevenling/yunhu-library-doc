# 左侧菜单栏设计

左侧菜单栏是一个拥有二级菜单的菜单栏。

通过后端获取菜单 json 信息后渲染。

点击任意一个二级菜单会向后端请求这个二级菜单的书籍信息，然后渲染到右侧的书籍信息显示区域。

# 右侧书籍信息显示区域

是一个卡片式信息。分页，每一页 10 个书籍信息。

可直接点击下载按钮进行下载书籍，下拉下载项，分为 pdf，mobi，epub，有对应的书籍文件才会显示并提供下载。

下载的书籍链接是存放到腾讯云对象存储中，只有一个 url，本地下载 pdf 会直接预览，因此下载时候设置在新窗口。

        /**
         * 导出
         * @param {} data 书籍的 url 
         * @param {*} fileName 文件名
         * @param {*} fileType 文件类型 eg: pdf, mobi, epub
         */
        function exportFile(data, fileName, fileType) {
            const url = data;
            const link = document.createElement("a");
            // 在新窗口进行点击
            link.target="_blank"
            link.style.display = "none";
            link.href = url;
            link.download = `${fileName}.${fileType}`;
            document.body.appendChild(link);
            link.click();
        }
