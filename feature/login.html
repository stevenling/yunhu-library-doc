# 登录

# 介绍

登录模块采用卡片式设计，左半边是一张图片，右半边是一个 <el-card> 标签。

卡片里面是 <el-form> 表单。

# 验证功能

使用 rules 对用户名和密码进行实时的验证。

    const rules = reactive({
      userName: [
        { required: true, message: "请填写用户名", trigger: ["blur", "change"] }
      ],
      password: [
        { required: true, message: "请填写密码", trigger: ["blur", "change"] },
        {
          min: 6,
          message: "密码至少要六位"
        }
      ]
    });

# 点击登录

  1. 先进行用户名和密码的校验
  2. 通过 axios 将表单数据发送到后端
  3. 接收到响应后，获取 token 值和当前用户名，然后存到 store 中，然后跳转到主页面。
