# 项目介绍

云胡图书馆是一个可以下载和在线阅读的电子书图书馆。

# 前期准备

  • Python 爬虫爬取书籍信息和图片
  • 将图片存到腾讯云对象存储中
  • 将书籍存到腾讯云对象存储中

# 功能

  • 登录、注册，注册需要邀请码，每一个邀请码有次数限制。
  • 二级菜单书籍分页显示并提供下载
  • 搜索下载书籍
  • 在线阅读
  • 复制文本
  • 导出摘抄图片
  • 高亮、下划线
